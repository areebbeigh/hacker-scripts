DESCRIPTION
-----------

hs-start

This command launches all the programs specified in the configuration when run.

"Why would I need this?"

You can use it to launch the programs you use on a daily basis, if they are more
than 2-3 programs this cmd could be useful.

CONFIGURATION
-------------

Here's an example configuration:

[hs-start]
	program1 = C:\Users\Areeb\Desktop\mIRC\mIRC.exe
	program2 = C:\Users\Areeb\Documents\hacker-scripts\hacker-scripts.sln

1. PROGRAMS

These generally are ".EXEs" or any other executables that you want to launch each
time the cmd/script is run.

USAGE
-----

hs-start [-h]

Required aruements:

None

Optional arguements:

--h, -h		Display this help documentation
